●mycel

node-based generative graphics studio

Wire source nodes (Noise, FlowField, Webcam) into filter nodes (Halftone, ASCII, Polar, Bloom…) and out to an Output. Drag nodes from the Library on the left, click to edit parameters on the right.

Wiring

Drag from a node's right port to another's left port. Right-click a node to delete. The Feedback node lets you make loops.

Sharing

Click ↗ Share to copy a remix link — anyone who opens it gets the exact graph and can tweak it. Try the Gallery for inspiration.

Shortcuts

Space play/pause   Delete remove selected node   right-click a node to delete

Export

Click PNG to save the current frame at the chosen resolution.
